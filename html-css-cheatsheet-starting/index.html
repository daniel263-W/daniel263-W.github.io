CSS Cheatsheet

Display and Positioning

Tag Name Description
z-index CSS z-index property The CSS z-index property specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
position: fixed; Fixed CSS Positioning When the CSS position has a value of fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling
display: block; Block elements Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
display: inline; Inline elements Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
display: inline-block; Inline-block elements Inline-block elements can appear next to each other, and can have their width and height manually adjusted.
position: absolute; absolute enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute.
position: relative; relative enables an element to be positioned relative to where it would have originally been on a web page.
